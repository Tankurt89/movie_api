My Flix

This Api will allow users to access a database of movies with the release date, and director.


Get:'/movies' returns a list of all the movies in the database.
Get:'/movies/:title' returns details on a specific movie.
Get:'/movies/genres/:genreName' returns genre information
Get:'/movies/directors/:directorName' returns the director, their birth/death year, and their bio.
Get:'/users/:Username' returns a specific user
POST: '/users' -- creates user -- request body requires field newUser
Put:'/users/:Username' -- username change -- request body requires field userId
DELETE: '/users/:Username' --deletes user by user id
POST: '/users/movies' -- adds a movie to user favorite list -- requires body field movieTitle
DELETE: '/users/:Username/movies/:MovieID' --deletes favorite movie off user profile
